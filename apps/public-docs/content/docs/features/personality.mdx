your Huginn AI has a personality that you control completely. Two files define that personality — **SOUL** and **IDENTITY** — and you edit them from the Personality page in your dashboard.

## SOUL vs IDENTITY

**SOUL** defines the AI's core character. Think of it as the personality layer that would be consistent regardless of who Huginn was talking to. This is where you set communication style, values, tone, areas of expertise, and how the AI approaches problems.

**IDENTITY** defines how Huginn relates specifically to _you_. This is where you write context about your life, your work, your preferences, and anything else you want the AI to know about you as an individual. The IDENTITY file is what transforms a generic assistant into one that feels like it knows you.

<Callout type="info">
  Both files are plain text — write them however feels natural to you. Many users write them in a
  loose prose style; others use bullet points. Huginn reads and applies whatever you write.
</Callout>

## Editing a file

<Steps>
<Step>
### Open the Personality page

Navigate to **Personality** in the top navigation bar. The editor opens with two tabs — **IDENTITY** and **SOUL** — with the IDENTITY tab active by default.

</Step>
<Step>
### Select the file you want to edit

Click the **SOUL** or **IDENTITY** tab to switch between files. If you have unsaved changes on the current file, you'll be asked to confirm before switching.

</Step>
<Step>
### Make your changes

Edit the text in the editor area. The content you write here is sent to Huginn as part of its system instructions on every conversation — so changes take effect immediately on your next chat.

</Step>
<Step>
### Add a reason (optional)

Below the editor is a **Reason** field. This is a short note to yourself describing what you changed and why. It's purely for your own reference — it helps you track the evolution of your personality files over time.

</Step>
<Step>
### Save

Click **Save Changes**. The save button is disabled until you make a change. Once saved, the new version is active immediately.

</Step>
</Steps>

## Version history

Every save creates a new version of the file — the previous version is never overwritten. This append-only approach means you can always see how your personality has evolved, and you're never at risk of accidentally losing something you wrote.

<Callout type="warn">
  The current editor does not display version history directly. The most recent version is always
  what Huginn uses. Earlier versions are preserved in the system for future reference.
</Callout>

## Tips for writing good personality files

**Be specific.** Vague instructions like "be helpful" are less effective than specific ones like "keep responses concise unless I ask for detail — I prefer bullet points over paragraphs."

**Write in natural language.** You don't need to use special syntax or formatting. Write the way you'd explain yourself to a new colleague.

**Update as you evolve.** Your needs and preferences change. Revisit SOUL and IDENTITY periodically and adjust them to reflect how you want Huginn to work for you today.

**Use IDENTITY for context.** The IDENTITY file is a great place to capture things like: your role and profession, ongoing projects, key relationships, communication preferences, things that frustrate you, and things you find valuable.
