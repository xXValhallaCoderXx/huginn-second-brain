# Huginn public docs

Welcome to the public documentation site for **Huginn Second Brain**.

This site is intended for content that is safe to share broadly, such as:

- product overviews
- onboarding guides
- setup instructions
- user-facing workflows
- public integration notes

## What belongs here

Use the public docs site for anything a customer, evaluator, or external collaborator should be able to read without extra context.

## What does not belong here

Keep the following out of this site:

- internal runbooks
- incident procedures
- secrets, tokens, or environment-specific values
- unpublished roadmap details
