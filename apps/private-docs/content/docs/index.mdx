# Huginn Engineering Documentation

Welcome to the internal engineering docs for **Huginn** — a personal AI assistant built as a Turborepo monorepo with TanStack Start, Mastra agents, and PostgreSQL.

## Key Concepts

- **One account → one personality → one memory → any channel** — Core design principle
- **Portfolio pattern** — Single database with schema isolation (`public`, `mastra`)
- **RequestContext injection** — Per-request dependency injection for agents
- **Append-only versioning** — Personality files never update, only insert new versions
- **Token encryption at rest** — All OAuth tokens encrypted with AES-256-GCM
